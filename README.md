# odoo-gen

**Odoo Code Generator** — CLI-утилита для генерации модулей и компонентов Odoo (18–19).

Позволяет быстро создавать структуру модуля: модели, views, меню, actions...

---

## Установка

### Установка для разработки
```bash
pip install -e .
```
### Или обычная установка
```bash
pip install .
```


## Использование

### Создать модуль с моделью и меню
odoo-gen -m <model_name> [path_to_project] 

### Справка по доступным параметрам:
```bash
odoo-gen --help
```

## Возможности

- Интерактивный CLI
- Генерация моделей и views
- Генерация и обновление меню
- Автоматическое обновление __manifest__.py
- Корректная работа с существующими файлами и структурой проекта


## Требования

- Python >= 3.11
- Зависимости:
  - click
  - jinja2
  - questionary
  - lxml
